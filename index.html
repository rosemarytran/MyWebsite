Tracking TV

Website Application

The website TrackingTV is aim to movie lovers, who watch/follow a lot of tv series. In this case he/she maybe can not keep track of what will be premiere soon. Therefore, this website provides customer with a basic function to “Add” their favorite series, show them in the calendar what series is coming up and also information on shows, cast like IMDB.

This website can be commercialize if it is granted permission to stream videos and subtitles officially (like for example Netflix).

I deployed this website on Azure using Virtual Machine(Ubuntu Server 14.04 LTS) and push my codes in(Ruby on Rails), make an end point( DNS of VM) and finally start server in daemon mode. Besides, I use One Drive to store my images for the website (using embed).
